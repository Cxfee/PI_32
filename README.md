Para simular esta aplicação no seu computador, instale o XAMPP e baixe a pasta PI32 dentro da pasta htdocs do XAMPP. No painel de controle do XAMPP ligue o servidor Apache e o MySQL. Abra o phpmyadmin e importe o banco de dados contido na pasta PI32. Abra o navegador e digite o endereço "localhost/PI32", o menu principal do site deverá aparecer e as funcionalidades poderão ser testadas normalmente.
A prova de conceito é a versão retrabalhada da jornada de usuário conforme o documento PI_grupo_32_REVISADO.pdf enviado no blackboard. 
Resumidamente a jornada consiste em criar uma conta, logar nesta conta, criar uma resenha e publicá-la além de obter as opções de compartilhamento pelo Whatsapp e de ler todas as resenhas publicadas.
